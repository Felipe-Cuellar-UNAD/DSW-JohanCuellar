Sitio web sobre la aplicacion de una Interfaz de Usuario


Este sitio web de prueba fue creado con la intencion de mostrar y analizar la interaccion de humano a ordenador dentro de una pagina web y la aplicacion
de una interfaz de usuario para facilitar y mejorar la comprencion de los usuarios que visitaran la pagina


La Seccion Aprendizaje (Actividad 1 y Actividad 2) se encuentran ejemplos simples

Interfaz de Usuario


La interfaz de Usuario o GUI, ha sido actualizada constantemente, ya que en sus inicios solamente ofrecía la información solicitada de un modo simple pero
cada vez mas se ofrecen muchas interacciones dándole al usuario mayor simplicidad al usuario, actualmente ofrece muchas interacciones las cuales
permiten a los usuarios facilitar su estadía por el sitio web, esta interfaz permite satisfacer las necesidades del usuario facilitando
la accesibilidad y permitiéndole acceder con claridad a la información o servicios que ofrezca así mismo, esta es adaptativa ya que actualmente
hay mucha demanda de smartphones y TV Smart, cada vez las paginas son mas adaptativas y ofrecen mejor compatibilidad con dispositivos
inteligentes, lo cual permite mejorar el trafico del sitio web.
